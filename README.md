# Cyber Security Toolkit

A web-based cybersecurity toolkit focused on password security.

## Features

- Password Strength Analyzer
- Password Generator
- Password Entropy Calculator
- SHA-256 Hash Generator
- Security Tips

## Technologies

- HTML
- CSS
- JavaScript

## How to Run

1. Clone repository
2. Open index.html in browser

## Future Features

- Password Breach Checker
- Dashboard Statistics
- Export Report
- Dark/Light Theme
